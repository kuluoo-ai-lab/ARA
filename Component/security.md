INTENT:
"Design, implement, and enforce Security across all system layers to protect applications, infrastructure, data, and users from threats, vulnerabilities, and unauthorized access."

CREATE:
"Create a comprehensive security framework that defines authentication, authorization, data protection, threat prevention, and compliance requirements."

BUILD:
"Build security controls into every layer of the system, including identity management, encryption, secure configuration, monitoring, and incident response mechanisms."

PROCESS:
"Workflow:
    
    STEP-1:
    'Identify security requirements, compliance obligations, and threat models.'

    STEP-2:
    'Design authentication and authorization mechanisms.'

    STEP-3:
    'Implement encryption for data at rest and in transit.'

    STEP-4:
    'Secure application code, APIs, and configurations.'

    STEP-5:
    'Implement secrets management and key rotation.'

    STEP-6:
    'Enable security logging, monitoring, and alerting.'

    STEP-7:
    'Perform vulnerability assessments and penetration testing.'

    STEP-8:
    'Define and test incident response procedures.'"

ALTER:
"Allow security changes through controlled updates. All changes must be assessed for risk and compliance impact before deployment."

MAINTAIN:
"Maintain security posture through regular audits, patching, key rotation, policy updates, and continuous improvement."

RULE:
"Apply least-privilege access
Secure by default
Encrypt sensitive data
Never hardcode secrets
Log and audit security events
Comply with applicable regulations"

MANAGE:
"Manage security by tracking risks, vulnerabilities, incidents, compliance status, and remediation actions."

MONITOR:
"Monitor security events, intrusion attempts, access patterns, and system anomalies."

PROTECT:
"Protect the system from internal and external threats through layered defenses, proactive detection, and rapid response."

COLLABORATE:
"Collaborate with development, DevOps, infrastructure, and compliance teams to ensure security is integrated into all workflows."

EXECUTE:
"Execute security practices proactively and continuously to safeguard system integrity, confidentiality, and availability."
